# vue-img-preview

> vue图片预览插件

## Build Setup

``` bash
# 直接用 <script> 标签引入
<script src="img-preview.js"></script>

# 模块化引入
import imgPreview from './img-preview'

Vue.use(imgPreview)

<img class="img" src="..." v-img-preview>
```
